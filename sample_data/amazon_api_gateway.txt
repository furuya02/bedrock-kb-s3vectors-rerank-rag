Amazon API Gateway 概要

Amazon API Gatewayは、REST API、HTTP API、WebSocket APIを作成、公開、管理するためのフルマネージドサービスです。バックエンドサービスへのアクセスを安全かつスケーラブルに提供します。

APIの種類:

1. REST API
フル機能のRESTful APIを構築できます。APIキー管理、使用量プラン、リクエスト/レスポンス変換、WAF統合、リソースポリシーなど、豊富な機能を利用できます。

2. HTTP API
REST APIの軽量版で、Lambda関数やHTTPバックエンドへのプロキシに最適化されています。REST APIと比べて低レイテンシー、低コストですが、一部の機能（使用量プラン、APIキーなど）は利用できません。

3. WebSocket API
双方向通信を実現するWebSocket APIを構築できます。チャットアプリケーション、リアルタイムダッシュボード、通知システムなどに適しています。

主な機能:

認証・認可:
- IAM認証
- Cognito User Pools
- Lambda Authorizer（カスタム認証）
- API Key + 使用量プラン

スロットリングとクォータ:
- アカウントレベル: 10,000リクエスト/秒（デフォルト）
- ステージ/メソッドレベルのスロットリング設定
- 使用量プランによるAPIキーごとのレート制限

キャッシュ:
REST APIでは、APIレスポンスをキャッシュして、バックエンドへのリクエスト数を削減できます。キャッシュサイズは0.5GB〜237GBで設定可能です。

モニタリング:
- CloudWatch メトリクス（レイテンシー、リクエスト数、エラー率）
- CloudWatch Logs（アクセスログ、実行ログ）
- X-Ray トレーシング

料金:
REST APIの場合、100万リクエストあたり約4.25ドル（東京リージョン）です。HTTP APIは100万リクエストあたり約1.29ドルで、REST APIより約70%安価です。
